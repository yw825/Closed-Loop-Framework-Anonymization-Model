# Closed-Loop Framework Anonymization Model

## Overview
This repository implements a closed-loop optimization framework 
to balance privacy protection and machine learning utility in healthcare data.

## Files
- `model_train.py` – Model training procedures
- `particle_swarm.py` – PSO optimization implementation
- `utils.py` – Helper functions
- `constants.py` – Global parameters
- `closed_loop.ipynb` – Experiment notebook
